Grain Texture

This entire page has a film-grain overlay applied via body::before. Notice how the texture sits over backgrounds, color swatches, the photograph, and the text — unified across the whole page.

The grain has pointer-events: none, so it never blocks clicks. It sits at z-index: 50 — above page content, but below modals and nav chrome (which typically use 1000+).
